# Audio Files

## Background Music
Add MP3 files to the `music/` folder:
- `Ashes.mp3`
- `Between The Lines.mp3`
- `Epic Quest.mp3`
- `Mystic Realms.mp3`
- `Mythic.mp3`

Recommended sources:
- Freepik: https://www.freepik.com/search?query=fantasy+background+music&format=search
- Pexels Music: https://www.pexels.com/search/music/ (filtered to royalty-free)
- YouTube Audio Library: https://www.youtube.com/audiolibrary
- Pixabay Music: https://pixabay.com/music/
